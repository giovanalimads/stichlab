# 🧶 STICHLAB

StitchLab é uma aplicação web gratuita que permite criar e editar gráficos de ponto cruz de forma simples e intuitiva. O foco do projeto é oferecer uma ferramenta acessível para iniciantes, sem necessidade de login, com uma interface moderna, fácil de usar e 100% gratuita.

## 🧠 Sobre

O StitchLab permite que o usuário crie gráficos personalizados de ponto cruz diretamente no navegador.

Na tela inicial, o usuário pode visualizar seus projetos salvos localmente ou iniciar um novo gráfico. Ao criar ou abrir um projeto, ele é levado ao editor, onde pode desenhar livremente em uma malha (grid), utilizando ferramentas como pincel e borracha.

O usuário pode selecionar cores (incluindo cores reais de linhas com código), pintar células clicando ou arrastando, e ajustar o gráfico conforme necessário. O sistema salva automaticamente os projetos no dispositivo, sem necessidade de conta.

Além disso, o projeto permite exportar os gráficos em formatos como imagem ou PDF, facilitando o uso fora da aplicação.

```bash

git clone https://github.com/giovanalimads/stichlab.git

# Entrar na pasta
cd stichlab

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```
